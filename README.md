# LoRa32 Node

A simple breakoutboard / antenna mount for those cool, inexpensive Heltec LoRa32.

## Features

 * DHT11 sensor
 * 18650 battery holder
 * Shitty Addon Connector
 * Piece to break out and use as an antenna mount
 * Designed in the Canton of Glarus
